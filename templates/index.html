Common Questions

Is this service really free?

+

Yes! Our basic shortening service is completely free to use without an account. We believe in open access to simple web tools.

Can I edit the destination URL later?

+

Currently, link destinations are permanent to ensure link integrity. We are working on a premium feature to allow mutable links.

How long do links last?

+

By default, links do not expire. However, you can set a custom expiration date (7 or 30 days) using the advanced options in the tool.
